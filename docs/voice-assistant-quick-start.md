## ðŸ†˜ Quick Troubleshooting

### "Elara isn't responding to my voice"
1. Check microphone permissions in browser
2. Make sure microphone is working (test in other apps)
3. Use Chrome or Edge for best compatibility
4. Speak clearly in a quiet environment

### "The voice sounds robotic"
- You might be using an older browser
- Try Chrome or Edge for the best natural voices
- Check that you have the latest browser version

### "Elara didn't understand me"
- Check the transcript to see what she heard
- Try rephrasing your request
- Speak a bit more clearly
- Reduce background noise
